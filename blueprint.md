# Blueprint: AI-Powered Nutrition App

## Overview

This document outlines the design, features, and development plan for a Flutter-based nutrition and wellness application. The app is designed to be beautiful, intuitive, and powered by generative AI to provide a personalized user experience.

## Design & Style

*   **Theme**: Modern, clean, and visually balanced, using Material 3 components.
*   **Color Palette**: A soft, inviting palette based on a primary seed color. We will prioritize light and dark mode compatibility.
*   **Typography**: Expressive and clear typography using `google_fonts` for a premium feel.
*   **Visuals**: Use of high-quality, relevant images and modern iconography to enhance user understanding. UI components will have a "lifted" feel with soft, multi-layered drop shadows.
*   **Interactivity**: Key interactive elements will feature a subtle "glow" effect to feel more responsive and engaging.

## Implemented Features

*   **Core Structure**: Basic Flutter application shell.
*   **Dependency Setup**: Added core packages for UI (`google_fonts`, `provider`), navigation (`go_router`), and functionality (`image_picker`, `http`).
*   **Basic Theming**: A preliminary theme structure is in place in `lib/main.dart`.

## Current Focus: Analytics & AI Report UI

My current goal is to build a beautiful and functional analytics screen and a visually appealing AI-driven report page.

*   **Analytics Screen**: I will create a new screen that visualizes user nutrition data (e.g., calories, macros) using soft, elegant charts and progress indicators from the `fl_chart` and `percent_indicator` packages.
*   **AI Report Page**: I will design a dedicated page to display insights generated by AI. The layout will be clean and easy to read, focusing on presenting the information clearly without overwhelming the user.
