# Calorie Tracker - приложение для подсчета калорий

## Обзор

Это приложение представляет собой полнофункциональный трекер калорий, созданный с использованием Flutter и Dart. Оно предоставляет пользователям возможность легко отслеживать свое питание, используя различные методы ввода, включая текстовый поиск, распознавание изображений и выбор из рецептов.

Дизайн приложения вдохновлен эстетикой iOS, с акцентом на чистый, минималистичный и удобный интерфейс. В нем используются современные иконки, плавная анимация и интуитивно понятная навигация, чтобы сделать процесс отслеживания калорий максимально простым и приятным.

## Реализованные функции

### Главный экран (`HomeScreen`)
*   **Сводка за день:** Отображает в виде круговой диаграммы общее количество потребленных калорий и оставшийся лимит.
*   **Карточки БЖУ:** Наглядные карточки для отслеживания белков, жиров и углеводов.
*   **Трекер воды:** Интерактивная карточка для отслеживания выпитой воды в течение дня. Позволяет добавлять "стаканы" и показывает прогресс до дневной цели.
*   **Список приемов пищи:** Разделы для завтрака, обеда, ужина и перекусов с возможностью добавления продуктов.
*   **Современный дизайн:** Чистый, минималистичный интерфейс в стиле iOS.

### Экран рецептов (`RecipesScreen`)
*   **Список рецептов:** Отображает список доступных рецептов в виде карточек с изображением, названием, калорийностью и временем приготовления.
*   **Фильтрация:** Возможность фильтровать рецепты по категориям.
*   **Поиск:** Поле для поиска рецептов по названию.

### Экран аналитики (`StatsScreen`)
*   **Переключатель периода:** Позволяет просматривать статистику за неделю или за месяц.
*   **Динамика калорий:** Линейный график, показывающий потребление калорий за выбранный период.
*   **Динамика веса:** Линейный график для отслеживания изменения веса.
*   **Среднее БЖУ:** Круговая диаграмма, показывающая среднее процентное соотношение белков, жиров и углеводов.
*   **Карточки прогресса:** Отображают ключевые показатели, такие как среднее количество шагов, вес, количество тренировок и выпитой воды.
*   **Отчет от AI:** Карточка с "умным" анализом и рекомендациями от нейросети (на данном этапе — макет).

## План

### Следующие шаги

*   **Реализация экрана "Профиль":**
    *   Создание базового макета экрана.
    *   Добавление аватара и имени пользователя.
    *   Поля для настройки личных данных (рост, вес, возраст, пол).
    *   Возможность установки ежедневных целей (калории, вода, шаги).
    *   Переключатель для смены темы (светлая/темная).
    *   Кнопка "Выход".
*   **Реализация функционала тренировок:**
    *   Создание экрана для добавления и просмотра тренировок.
    *   Возможность указывать тип, продолжительность и сожженные калории.
    *   Интеграция данных о тренировках в общую статистику.
